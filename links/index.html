Научные работы и статьи

8. Основы проектирования реляционных баз данных

Авторы: Кириллов В.В.

Описание: Учебное пособие содержит подробное изложение вопроса проектирования реляционных баз данных, что поможет выбрать наилучший метод проектирования, исходя из конкретных потребностей своей рабочей среды.

9. Технологии статического и динамического анализа уязвимостей программного обеспечения

Авторы: Аветисян А., Белеванцев А., Чукляев И.

Описание: В статье описано практическое использование статического анализа для поиска ошибок и уязвимостей в исходном коде.

10. Инкрементальный статический анализ кода на основе разложения отношений

Авторы: Терехов А.Н., Булычев Д.Ю.

Описание: В статье рассматривается подход к инкрементализации статического анализа кода, позволяющий выполнять эффективный пересчет результатов анализа при малых изменениях кода

11. Использование зависимостей для повышения точности статического анализа программ

Авторы: Глухих М.И., Ицыксон В.М., Цесько В.А.

Описание: Статья посвящена разработке методов анализа зависимостей с целью повышения точности статического анализа. Рассмотрены причины, ведущие к снижению точности абстрактной интерпретации при обнаружении дефектов в программном коде.

12. Dynamic Analysis of SQL Statements for Data-Intensive Applications Reverse Engineering

Авторы: Cleve A., Hainaut J-L.

Описание: В данной статье рассматривается проблема динамического анализа SQL-запросов в контексте программного обеспечения и обратного проектирования баз данных. В нем исследуется использование методов динамического анализа, таких как трассировка на основе аспектов и анализ трассировки SQL для извлечения неявной информации о поведении программы и структуры базы данных.

13. JDBC Checker: A Static Analysis Tool for SQL/JDBC Applications

Авторы: Gould C., Su Z., Devanbu P.

Описание: В статье описывается инстурмент для статического анализа приложений, работающих с СУБД посредством SQL-инструкций, и принцип его работы.

14. Automatic SQL Tuning in Oracle 10g

Авторы: Dageville B., Das D., Dias K.

Описание: В публикации описан процесс автоматизированной оптимизации SQL-интструкций, определяющих структуру базы данных и запросов к ней, в контекте СУБД Oracle.

15. Семантико-синтаксический анализ естественных языков

Авторы: Смирнов И., Шелманов А.

Описание: Рассмотрены задачи семантико-синтаксического анализа текстов на естественных языках. Приведен обзор подходов и методов синтаксического и семантического анализа текстов. Сделаны выводы о применимости существующих подходов к разработке методов семантико-синтаксического анализа текстов.

16. Compiler Construction: Principles and Practice

Авторы: Louden K.

Описание: Публикация раскрывает основные принципы построения компилятора и инструменты, позволяющие использовать эти концепции.

17. Теория синтаксического анализа, перевода и компиляции

Авторы: Ахо А., Ульман Дж.

Описание: Cодержит основной математический аппарат, краткий обзор процесса компиляции, начала теории синтаксически управляемого перевода и обстоятельное изложение методов синтаксического анализа. Рассмотрены и систематизированы почти все известные алгоритмы разбора.

18. Abstract Syntax Tree Implementation Idioms

Авторы: Jones J.

Описание: В статье описаны проблемы и решение задачи представления конструкций исходного кода, полученные в результате анализа, на языке реализации транслятора.

19. Генератор синтаксических анализаторов для неоднозначных контекстно-свободных грамматик

Авторы: Григорьев С.

Описание: В работе описаны возможные алгоритмы для решения проблемы анализа языков имеющих неоднозначную грамматику (набор специфических диалектов).

20. Базы данных. Основные понятия, модели данных, процесс проектирования
